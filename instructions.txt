Вы находитесь в режиме изменения дерева. Он создан, чтобы редактировать поля с данными и сливать вместе деревья.
Для выхода из этого режима введите 'stop'.
Синтаксис команд и их описние: 
    1) команды типа show:
        -tree - вывод дерева, с которым сейчас работает программа
        -file *название бинарного файла* - вывод дерева, построенного по файлу 
        -differences *название первого элемента* *название второго* - выод сходств и различий элементов
        -path *название элемента* - вывод признаков элемента

    2) команды типа select:
        для выбора элемента необходимо ввести:
        select *название элемента*

        -add before *название нового элемента* - добавление нового элемента до выбранного.
        Если выбранный элемент был левым у предшественника, то при добавлении элемента новый элемент станет левым для предшественника,
        а выбранный станет левым для нового. Правая ветка нового элемента инициализируется NULL. (аналогично для правой стороны)
        (добавление элементов до корня невозможно)
        -add left/right *название элемента* - добавляет левый или правый элемент после выбранного. Если у выбранного элемента есть левый последователь,
        то при добавлении левого элемента последователь становится левым элементом. Правая ветка нового элемента инициализируется NULL. (аналогично для правой стороны)
        -delete - удаление выбранного элемента. Возможно удаление только элементов без последователей.
        -change *новая информация* - записываетв поле данных новую информацию
        -merge *название файла с поддеревом* - если данные выбранного элемента совпадают с данными в корне поддерева, то к выбранному элементу прикрепляется поддерево.
        (у выбранного элемента не должно быть последователей)
        

    пример команды:
    show differences student human
    show file Akinator.bin

Структура вывода дерева:

    root
        |__(NO brunch)
        |            |
       ...          ...
        |__(YES brunch)
                      |
                     ...
